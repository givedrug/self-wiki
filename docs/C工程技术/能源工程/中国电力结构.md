中华人民共和国的能源结构长期以煤炭为主，水力发电也有一定比重。改革开放后，能源需求随着经济发展增长，并正朝向高效率低污染的石油气发电型式过渡，与此同时，以风电、核电、太阳能为代表的新能源正在快速增长。中华人民共和国可再生能源部分已位居世界第一。
